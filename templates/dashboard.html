ℹ️ How This Tracking Works

🛡️ No Third-Party Scripts

This dashboard uses zero external analytics (no Google Analytics, no cookies from ad networks). All data stays on this server.

🪝 Flask Request Hook

The @app.before_request decorator intercepts every incoming request and logs it to a local SQLite database before the route handler runs.

🗂️ SQLite Storage

All pageviews, category hits, and clickstream paths are stored in data/analytics.db — a lightweight, file-based database with no external dependencies.

🔗 Session Clickstreams

Flask's session cookie tracks an ordered list of pages each visitor navigates. This reveals entry points, exit points, and common navigation patterns.
